## Description

Simple LAN multicast messaging application made with Svelte + NodeJS and Serial Port support


## Requirements

Must have installed:
- https://nodejs.org/


## Installation

    $ git clone https://github.com/manuelgm0518/multicast-chatrooms
    $ cd multicast-chatrooms
    $ npm install


## Running the app

    # Terminal 1
    $ cd client
    $ npm run dev
    
    # Terminal 2
    $ cd server
    $ npm run dev
